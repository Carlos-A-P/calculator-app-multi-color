Keypad instructions

To navigate around the keypad, use the W, A, S, D keys or the arrow keys on the keyboard and press space to activate the key

  • The W key allows you to move up
  • The A key allows you to move left
  • The S key allows you to move down
  • The D key allows you to move right

Lastly, you can use the number keypad to input number values and use the Enter button to compute
